# 코딩 연습 - 연산자 오버로딩 (1)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 단항 연산자

`Counter` 구조체를 정의하고, 단항 `-` 연산자와 `++` 연산자를 오버로딩하세요.
`Counter(5)`를 생성한 후 `-c`로 음수 값을, `c++` 후 값을 출력하세요.
`ToString()`을 재정의하여 `Value` 값을 문자열로 반환하세요.

**실행 결과**

```
-5
6
```

---

## 2. 이항 연산자

`Fraction` 구조체를 정의하고, `+`와 `*` 연산자를 오버로딩하세요.
`Fraction(1, 2)`와 `Fraction(1, 3)`을 생성하여 덧셈과 곱셈 결과를 출력하세요.
`ToString()`을 재정의하여 `분자/분모` 형식으로 반환하세요.

**실행 결과**

```
5/6
1/6
```

---

## 3. 비교 연산자

`Money` 구조체를 정의하고, 
`==`, `!=`, `<`, `>` 비교 연산자를 오버로딩하세요.
`Money(1000, "KRW")`와 `Money(2000, "KRW")`
를 생성하여 동등 비교와 대소 비교 결과를 출력하세요.
`Equals()`와 `GetHashCode()
`도 함께 재정의하세요.

**실행 결과**

```
False
True
```

---

## 4. 복합 대입 연산자

`Vector2` 구조체를 정의하고, `+` 연산자를 오버로딩하세요.
`Vector2(1, 2)`를 생성한 후 `+= new Vector2(3, 4)`를 수행하고 결과를 출력하세요.
`ToString()`을 재정의하여 `(X, Y)` 형식으로 반환하세요.

**실행 결과**

```
(4, 6)
```

---

## 5. 암시적 변환

`Celsius` 구조체를 정의하고, 
`double`과의 암시적 변환 연산자를 양방향으로 오버로딩하세요.
`Celsius temp = 36.5;`로 암시적 변환 후, 
`double value = temp;`로 다시 변환하여 출력하세요.

**실행 결과**

```
36.5
```
